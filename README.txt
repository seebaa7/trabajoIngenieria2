Repositorio git generado para el sistema a desarrollar
para la catedra de Ingenieria de software II,
 dictada en la universidad nacional de La Plata (UNLP). 2016.

Utilizaremos PHP+Javascript como lenguajes de programacion, contenidos en HTML5.
MySQL como motor de la base de datos, y Jquery y CSS para manejar los estilos.
